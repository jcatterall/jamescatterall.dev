## Generative / Visual Tools

- Colour palette extractor — upload image, extract dominant palette, export as CSS variables / Tailwind config / hex codes and swatch SVG
- QR Code Studio — styled QR codes with dot shapes, colours, and embedded logos (sibling to Barcode Studio)
- ASCII art generator — convert uploaded images to ASCII/Unicode block art, adjustable density and character sets, export as SVG or plain text
- Font specimen maker — paste a Google Font name, preview across weights and sizes with live type-scale controls, export specimen card as SVG

## Developer Utilities

- Regex playground — live regex tester with match highlighting, named groups visualised, cheat sheet sidebar
- JSON diff viewer — paste two JSON blobs, see structural diff with added/removed/changed nodes highlighted
- Cron expression builder — visual builder with plain-English schedule description and next-run preview
- Timestamp converter — paste any timestamp (Unix, ISO, relative), convert between formats and timezones instantly

## Playful / Generative

- Pomodoro timer with keyboard shortcuts and desktop notifications
- Pixel grid editor — tiny pixel canvas (16×16, 32×32) with a limited palette, export as SVG or PNG
